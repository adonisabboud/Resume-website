Arabic Maqām Builder (53-EDO)

Interactive web app for exploring Arabic maqāmāt using a 53-tone equal temperament (Turkish comma system). Implements microtonal intonation (Rāst / Bayātī / Sīkāh half-flats) and renders sound via the Web Audio API with keyboard playback, drones, and transposition.

Tech stack: JavaScript, Web Audio API, HTML/CSS, GitHub Pages.
